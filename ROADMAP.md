# Roadmap

Built with **GAP OS** using **The Signal Hunter Framework**.

Tarifio is designed to simplify tariff calculations, improve trade decision-making, and make customs costs more transparent for global businesses.

---

# Phase 1 — Core Foundation (Current)

## Goal

Build a fast, transparent, open-source tariff calculator.

### Features

- Public open-source release
- Basic tariff calculator
- Country of Origin selection
- Destination Country selection
- HS Code input
- Product Value input
- Basic duty calculation
- Responsive UI
- MIT License
- Documentation

---

# Phase 2 — Product Maturity

## Goal

Provide practical calculations for day-to-day import planning.

### Features

- VAT / GST calculation
- Customs duty breakdown
- Additional duty support
- Trade agreement detection
- Multi-currency support
- Exchange rate integration
- CSV export
- Printable calculation reports
- Bulk calculations
- Saved calculation history

---

# Phase 3 — Intelligence & Automation

## Goal

Transform tariff calculation into decision support.

### Features

- Real-time tariff datasets
- REST API
- AI-assisted tariff recommendations
- Country comparison engine
- Alternative sourcing analysis
- Scenario simulation
- Cost optimization suggestions
- Performance optimization
- Community contributions

---

# Phase 4 — GAP OS Integration

## Goal

Connect tariff intelligence with the broader GAP OS ecosystem.

### Features

- Integration with HA.RS.QO
- Automatic HS Code lookup
- Integration with Netto
- Integration with Proveni
- Integration with Compila
- Cross-tool workflows
- Shared trade intelligence
- AI feedback learning
- Unified trade decision engine

---

# Long-Term Vision

Tarifio is evolving beyond a tariff calculator.

The long-term objective is to become an open trade intelligence engine capable of explaining *why* duties apply, identifying optimization opportunities, and supporting better sourcing decisions across global supply chains.
